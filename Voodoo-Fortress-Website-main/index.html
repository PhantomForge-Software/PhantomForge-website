Voodoo Fortress

Voodoo Fortress is a game that we are working on with our friends at Phantom Forge. It's an horror game where you can explore and interact with various environments while battling against monsters in dark places. The main goal of this project is to create a fun experience for players who enjoy exploring and encountering new things every day.
